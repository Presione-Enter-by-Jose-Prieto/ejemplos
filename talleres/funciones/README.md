# Taller de funciones

Desarrolla los siguientes ejercicios como estrategia de profundización, relacionando los conceptos básicos de funciones, parámetros, argumentos y valores de retorno.

## Enunciados

1. Desarrolla una función que reciba cinco números enteros como parámetros y muestre por pantalla el resultado de sumarlos (tipo procedimiento, sin valor de retorno).

2. Desarrolla una función que reciba cinco números enteros como argumentos y devuelva el resultado de sumarlos. Asigna el resultado de invocar la función con los números `2, 5, 1, 8, 10` a una variable de nombre `$temporal` y muestra el resultado.

3. Desarrolla una función que reciba como parámetros el radio de la base y la altura de un cilindro, y devuelva su volumen. Ten en cuenta que el volumen se calcula como:

   > **Volumen = π × radio² × altura**

   Utiliza la constante `M_PI` o el valor aproximado `3.1416`.

4. Diseña una función que reciba un número entero e indique por pantalla si dicho número es par o impar.

5. Desarrolla una función que incremente en uno un número pasado como argumento y muestre el resultado.

6. Crea una función que reciba un número entre `1` y `12` y retorne el nombre del mes correspondiente.

7. Desarrolla una función que calcule el IVA aplicable a un producto. La función debe recibir el total del producto y el porcentaje de IVA a aplicar; si no se proporciona el IVA, deberá tomar `10` como valor por defecto.
