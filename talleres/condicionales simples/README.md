# Taller de condicionales simples

Desarrolla los siguientes ejercicios como estrategia de profundización, relacionando los conceptos básicos de condicionales, ciclos, arrays, funciones y `math`.

## Enunciados

1. Desarrolla un algoritmo que genere un número aleatorio de 1 a 100 y, por medio de una condicional, determine si el número es menor que 50 o mayor que 50.
2. Desarrolla un algoritmo que, a partir de la cantidad de horas trabajadas de una persona y las horas de trabajo normal, determine si tuvo horas extras y, si las tuvo, cuántas fueron.
3. Dados dos números enteros, determina cuál es el mayor por medio de una condicional y realiza una potencia usando el número menor como exponente.
4. Realiza el punto número 3 usando el operador ternario.
5. Dado un número entero, determina por medio de una condicional si es negativo o no (menor que cero).
6. Dados dos números enteros, determina si uno es múltiplo del otro por medio del operador módulo.
7. A partir de una nota, determina si el estudiante aprobó o no la clase (`0` a `6` = perdió, `7` a `10` = aprobó).
