# Ejercicios con ciclos con arrays

## Enunciados

1. **Desarrollar un programa** que almacene en un array los 10 primeros números pares. Imprímelos, cada uno en una línea.

2. **Desarrollar un programa** que imprima el siguiente array usando la estructura cíclica `foreach`:

   ```php
   $arrayName[4] = 80;
   $arrayName[9] = "Hola";
   $arrayName[22] = 99;
   $arrayName[156] = "Mundo";
   ```

3. **Crear un array asociativo** para los siguientes datos:
   - **Nombre:** Juan Orozco
   - **Dirección:** Crra 48 A # 22
   - **Teléfono:** 3215489621

4. **Desarrollar un programa** que, por medio de tres arrays, defina la cantidad de datos por array y realice lo siguiente:
   - almacenar números pares aleatorios entre `0` y `100` en el primer array
   - almacenar números impares aleatorios entre `0` y `50` en el segundo array
   - almacenar la suma de los valores de ambos arrays en el tercer array

5. **Desarrollar un programa** que, dado un array de números enteros entre `0` y `50`, halle el número mayor almacenado.
   - Puedes utilizar variables auxiliares.

6. **Desarrollar un programa** que, por medio de un array de tipo `float`, permita almacenar el peso de 5 personas para posteriormente:
   - determinar el promedio del peso de las 5 personas
   - determinar cuántas personas superan el promedio del peso
   - determinar cuántas personas están por debajo del promedio del peso

7. **Desarrollar un programa** que, dados dos grupos (`Grupo A` y `Grupo B`), cada uno con 10 estudiantes y una nota final de la materia por estudiante, permita determinar:
   - el promedio general de cada grupo
   - el promedio general de ambos grupos
   - el grupo con mejor promedio

8. **Desarrollar un programa** que, dada una palabra, divida todos sus caracteres y los almacene en las posiciones de un array del tamaño de la palabra.

   Ejemplo:
   - `"Colombia"` tiene 8 caracteres, por ende, el array debe ser de tamaño 8.
