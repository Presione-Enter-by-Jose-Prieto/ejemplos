# Taller de Math

Desarrolla los siguientes ejercicios como estrategia de profundización, relacionando los conceptos básicos de funciones matemáticas, generación de números aleatorios y operaciones trigonométricas.

## Enunciados

1. Desarrolla un algoritmo que, por medio de un ciclo `for` de 10 iteraciones, almacene en un array 10 números generados aleatoriamente entre `0` y `50`, y determine cuál es el número mayor del array.

2. Halla el área de un círculo con radio `7`. Ten en cuenta que el área se calcula como:

   > **Área = π × radio²**

   Utiliza la constante `M_PI` o el valor aproximado `3.1416`.

3. Realiza una operación trigonométrica del grado de dificultad que consideres, usando las funciones matemáticas disponibles (`sin`, `cos`, `tan`, etc.).

4. Calcula la hipotenusa de un triángulo rectángulo cuyos catetos son `A = 5 cm` y `B = 6 cm`. Ten en cuenta que la hipotenusa se calcula como:

   > **c = √(A² + B²)**
